Description
===

A plugin that enables all of the futusign supplemental plugins to be updated from the multisite Network Admin site; without having network activate them. An add on to the [futusign](https://github.com/larkintuckerllc/futusign-wordpress)
digital signage management software.

Installation
====

The latest plugin is available for download from the *release* tab.

Upload and network activate the plugin via the WordPress Plugins multisite network admin screen.

Usage
====

Like plugins installed from the WordPress Plugin Directory, the other futusign supplemental commerical  plugins (not network activated) will report when new versions are available and can be upgraded with one click.
